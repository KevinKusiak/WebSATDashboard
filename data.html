Project Data


I initially tried to find real SAT scores on the District 211 website, but I was unable to find the actual SAT scores. After researching the SAT score structure on the web, I manually created the SAT score dataset.

Creating SAT Scores
  1. I created a spreadsheet for SAT section and individual tests in Excel first. I also added columns for school names, years, and geolocations for possible mapping. The table below reflects what the Excel spreadsheet data looked like.
  2. Next I used randbetween() function in Excel to generate random English and Math section scores as well as the years. The subtotal and total columns had formulas and updated on their own.
  3. I also looked up the addresses of the 5 schools in Google maps and recorded their geographical locations.
  4. Then I used an online CSV to JSON converter to convert my Excel data into an array of JSON objects. http://www.convertcsv.com/csv-to-json.htm
  5. The output file was of type .json. After downloading the file, I changed its format to .js and assigned the array of JSON objects to a JavaScript variable.
  6. Then I referenced this data file on every page that displays the data.
  7. This data file contains 250 rows of SAT scores for 5 District 211 schools for years 2014-2017.
