Select your Favorite School


This page allows for selecting a favorite school. When a favorite school is selected, the home page will by default calculate and display all the statistics and data views for that school. Otherwise, the same will happen for all the schools.

To select your favorite school:
  1. Select any of the 5 district 211 schools from the list
  2. Click the Save Favorite button to store the school name in the local browser storage.
  3. The application menu will show the school name next to My School
  4. All data views on the home page will by default execute for the favorite school
  5. Please note that this feature requires a web server when using an Internet Explorer. Please use Chrome or Firefox instead.
To remove your favorite school:
  1. Click the Delete Favorite button to remove the school name from the local browser storage.
  2. The application menu will no longer show the school name next to My School
  3. All data views on the home page will by default execute for all schools now.
